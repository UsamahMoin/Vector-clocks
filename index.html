How to read the model

Logical time tracks knowledge, not elapsed seconds.

A vector component counts events known from one process. If every component of A is less than or equal to B, and at least one is smaller, then A could have influenced B. If neither vector is component-wise less than the other, the events are concurrent: the system has no causal evidence that either happened first.

[1, 0, 0] < [1, 2, 0] causal
[2, 0, 0] ∥ [0, 1, 0] concurrent
